# NFC Badge Reader - Web Mock Version

A web-based mock NFC badge reader app that works on **any device** - iPhone, Android, Mac, Windows, etc. Perfect for testing and demoing the badge verification concept.

## 🚀 Quick Start

### Online (Easiest)
1. Visit: https://masonfkelley-dotcom.github.io/NFC-Reader-/
2. Click **"✓ Authorized Card"** or **"✗ Unauthorized Card"** buttons
3. Watch the app respond!

### Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/masonfkelley-dotcom/NFC-Reader-.git
   cd NFC-Reader-
   ```

2. Open `index.html` in your browser
   - Double-click the file, OR
   - Right-click → Open with → Browser

## ✨ Features

✅ **Authorized Card Simulation**
- Shows "Badge Verified Welcome!" in green
- Background turns green
- Phone vibrates (if supported)
- Auto-resets after 3 seconds

✅ **Unauthorized Card Simulation**
- Shows "Badge Unverified Access Denied" in red
- Background turns red
- Phone vibrates in pattern (if supported)
- Auto-resets after 3 seconds

✅ **Works Everywhere**
- iPhone, iPad, Mac
- Android, Windows, Linux
- Desktop, tablet, mobile
- Any modern browser

✅ **No Installation Needed**
- Just open in browser
- No app store
- No build process

## 🎮 How to Use

### Button Controls
- Click **"✓ Authorized Card"** - Simulate approved badge
- Click **"✗ Unauthorized Card"** - Simulate rejected badge
- Click **"Reset"** - Go back to home screen

### Keyboard Shortcuts (Desktop)
- Press **1** - Simulate authorized card
- Press **2** - Simulate unauthorized card
- Press **0** - Reset home screen

## 📱 On iPhone

1. Open this page in Safari
2. Tap the **Share** button
3. Select **"Add to Home Screen"**
4. Now it works like an app!
5. Click buttons to test
